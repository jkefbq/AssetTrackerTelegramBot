# AssetTracker (Телеграм-бот)
>Телеграм-бот помощник в управлении активами и торговле. Ты можешь добавлять, удалять активы, смотреть статистику по своему портфелю активов, а также -в планах- торговать с нейросетью. Gemini 2.5-flash дает индивидуальные советы по торговле исходя из твоего портфеля.
